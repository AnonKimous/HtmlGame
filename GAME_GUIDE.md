# 游戏教程：游乐园奖品射击摊

这份教程按“第一次打开游戏 → 打一局 → 进商店 → 解锁隐藏内容”的顺序写，适合直接放在 GitHub 仓库里。

---

## 1. 开始游戏

打开页面后，游戏会先显示全屏提示。点击“点击全屏开始”后即可进入主界面。手机端建议横屏游玩；如果浏览器顶部地址栏影响体验，可以把网页添加到主屏幕后再打开。

主菜单有四个主要入口：

| 按钮 | 作用 |
|---|---|
| 继续游戏 | 直接开始一局 |
| 新的开始 | 清空当前本地存档并重新开始 |
| 退出 | 尝试离开当前页面 |
| 国王的小店 | 不打一局，直接进入商店 |

主菜单中间的国王可以拖动。拖动不会影响战斗数值，只是国王会表达一点意见。

---

## 2. 一局游戏的目标

你的目标是在限定时间内击落尽量多的奖品，获得分数和彩票。

游戏中有三条传送带，奖品会从不同方向移动。越远的传送带通常奖品更小、更难打，但结算时会有深度分数加成。普通奖品包括小熊、鸭子、皮球、机器人、星星和皇冠。

基础分值如下：

| 奖品 | 基础分 |
|---|---:|
| 皮球 | 50 |
| 鸭子 | 60 |
| 小熊 | 80 |
| 机器人 | 120 |
| 星星 | 140 |
| 皇冠 | 200 |
| 国王 | 456 |

国王的 456 分来自 `168 * e` 向下取整，也可以视为一个彩蛋。国王通常会先伪装成其他奖品，第一次命中会破除伪装；真正击中国王时，他有极高概率闪避。

---

## 3. 基础操作

| 操作 | 效果 |
|---|---|
| 鼠标左键 / 触屏点击 | 普通射击或点击界面按钮 |
| 按住鼠标 | 适用于连发气枪、火星枪等自动武器 |
| 空格 | 根据当前界面执行快捷操作：开始、进商店、开始下一局或加速对话 |
| `M` | 开关音乐 / 音效 |
| 右键 | 在国王对话中加速或跳过 |

基础气枪有 8 发弹匣。弹匣打空后会自动装填，装填期间不能继续开火。

---

## 4. 分数、彩票与连击

击中奖品会获得分数和彩票。分数会受到这些因素影响：

1. 奖品基础分。
2. 传送带深度加成。
3. 当前连击加成。
4. 难度修正。
5. 特殊事件或国王结算倍率。

彩票大致根据本次得分折算；购买“幸运挂件”后，彩票收益会提高 50%。购买“稳定挂件”后，命中判定会略微放宽，更容易打中奖品。

连击越高，分数增长越快。长时间没有命中会断连。

---

## 5. 狂热模式

当本局累计的有效得分达到狂热阈值时，会进入狂热模式。狂热时：

- 奖品生成更密集。
- 传送带移动更快。
- 普通弹药限制基本取消。
- 部分武器会获得专属强化。
- 国王会在 6 到 9 秒后混入奖品。

狂热结束后会进入约 10 秒冷却。狂热期间达到一定连击，还可以获得少量狂热时间延长。

“狂热沙漏”的作用是：如果倒计时结束时你仍处于狂热中，本局不会马上结算，而是会续到狂热结束。

---

## 6. 商店与武器

结算后可以选择“进入商店”或“返回主菜单”。商店使用彩票购买商品，已拥有的武器可以再次点击来装备。

| 商品 | 价格 | 类型 | 说明 |
|---|---:|---|---|
| 基础气枪 | 0 | 武器 | 8 发弹匣，点击射击，空弹后自动装填 |
| 幸运挂件 | 500 | 挂件 | 彩票收益 +50% |
| 稳定挂件 | 800 | 挂件 | 命中判定略微放宽 |
| 连发气枪 | 2,000 | 武器 | 14 发弹匣，按住鼠标可低速连发 |
| 狂热沙漏 | 3,000 | 挂件 | 若时间结束时仍在狂热，续时到狂热结束 |
| 弧弹枪 | 7,000 | 武器 | 普通模式拖线后松开，扇形打出 8 发；狂热时自动大弦 12 发 |
| 皇家权杖 | 12,000 | 武器 | 点击挥击，短距离横扫；狂热时召唤 4 把护卫枪 |
| 火星枪 | 20,000 | 武器 | 无需换弹，自带低速扫射；狂热时定期发射火星核弹 |

### 弧弹枪怎么用

普通模式下，按住鼠标从起点拖到终点，形成一条“弦”。松开后，子弹会根据“起点 → 枪口 → 终点”的夹角方向扇形发射。如果拖得太短，会提示“弦太短”。

### 皇家权杖怎么用

皇家权杖没有弹匣。每次点击都会挥击一次，最多可以扫到附近 4 个目标。命中国王时，权杖会被视为多次攻击，但国王仍可能闪避。狂热时，皇家权杖会召唤 4 把护卫枪自动援护。

### 火星枪怎么用

火星枪无需装填，普通模式按住鼠标会低速扫射。狂热时，按住鼠标会以目标位置发射火星核弹，范围大，但国王依然有特殊闪避逻辑。

## 7. 债务与“抢货”机制

如果彩票不足，普通情况下无法购买商品。部分高价商品在你已经拥有至少约 80% 价格的彩票时，会触发“赊账/抢货”相关逻辑：多次尝试后，可能抢到商品，但会欠国王一笔带利息的债。

欠债时：

- 商店会优先催债。
- 不能正常继续购买未拥有商品。
- 点击商店右侧的国王可以还债。
- 债务还清后，商店功能恢复正常。

---

## 8. 国王与隐藏机制

国王是本游戏的核心彩蛋目标。他会伪装成普通奖品混入奖品堆，第一次命中只会破除伪装。被揭露后，国王有极高概率闪避普通攻击。

击落国王后会进入国王对话。对话期间可以右键或按空格加速。对话结束后，会触发全场结算效果。

---

## 9. 成就界面：国王家的黑暗

购买所有武器后，在商店右上角可以进入“国王家的黑暗”。里面有四个栏目：

| 栏目 | 内容 |
|---|---|
| 玩偶图鉴 | 查看普通奖品和国王的分值说明 |
| 国王对话 | 按情绪分类查看国王的台词 |
| 游戏成就 | 查看武器收集、彩票、单局分数和秘密收集等成就 |
| 私密小对话 | 查看已经解锁的私密问题 |

国王对话的五种情绪分类名称是：恐虐、欢愉、黑暗、卖垢、奸奇。

私密小对话比较特殊：国王不愿意直接说。已经解锁的问题需要点击一次让国王确认，再点一次才会显示答案。

---

## 10. 小祥超市：隐藏秘密商店

小祥超市是隐藏入口。开启方法：

1. 先把大奖商店里的所有商品都买完。
2. 回到普通商店界面。
3. 连续点击左上方“大奖商店”的 headline 区域 20 次。
4. 成功后会进入“小祥超市”。

小祥超市规则：

- 每次抽纸条价格为 1,000 到 5,000 彩票，价格看国王心情随机变化。
- 纸条抽取没有总次数上限，已经揭示过的秘密也可能再次抽到。
- 不能赊账。
- 最多同时持有 7 张纸条，防止 UI 溢出。
- 抽到纸条后，需要把纸条拖给右侧国王，才会消耗纸条并揭示对应私密小对话。

---

## 11. 存档与重置

游戏进度保存在浏览器本地的 `localStorage` 中，包括彩票、债务、最高分、已购商品、当前武器和已解锁秘密。

重置方式：

- 在主菜单点击“新的开始”，会清空当前游戏存档。
- 也可以在浏览器里清除该网页的站点数据。

注意：如果你换浏览器、换设备、开无痕模式，存档不会自动同步。

---

## 12. 新手建议

1. 先用基础气枪熟悉奖品速度和传送带层级。
2. 优先购买“幸运挂件”，这样后续彩票增长更快。
3. 再买“稳定挂件”，减少空枪带来的连击损失。
4. 连发气枪适合刷普通奖品；弧弹枪适合打扇形区域；皇家权杖和火星枪更像后期玩具。
5. 狂热时不要只盯普通奖品，注意国王混入提示。
6. 商店被买空后，再去寻找隐藏入口和私密小对话。
